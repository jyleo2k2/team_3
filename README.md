메인에서 1, 2, 3 입력받은 후 게임 실행, 4 입력 받으면 랭킹, 5 입력 받으면 종료

숫자 하나 입력받을 때 : 다음 함수 사용

def one_number_only(prompt):
    while True:
        num = input(prompt)
        try:
            num = int(num)
            if num in [1,2,3,4,5]:
                return num
            print("1 ~ 5 까지의 숫자를 입력해주세요.")
        except:
            print("숫자를 입력해주세요.")

2. 점수 배정

2-1. 10초 : 100점 만점, 10초와 본인이 누른 타이밍의 차이를 계산해서 이를 100배 한 값 점수로
(~9초 and 11초~) : 0점
(10.5초) : 50점
(10초) : 100점

2-2. 숫자야구 : 100점 만점, (11 - 시도횟수) * 10 점
(1번) : 100점
(3번) : 80점

2-3 홀짝 : 100점 만점, 10게임 한 뒤 맞춘 게임 수 x 10
(6게임) : 60점
(10게임) : 100점
